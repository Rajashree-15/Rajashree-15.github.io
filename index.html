ALPHA-dq Platform
Indian Institute of Science · Board Bring-Up & Firmware Stack

  • Developed bottom-up peripheral drivers in C for UART, ADC, PWM, SPI, and GPIO using memory-mapped register access, bringing up one subsystem at a time from reset

P.02
Custom USB Bootloader
IISc · ATmega328P

Engineered a custom bootloader for the ATmega328P controller enabling MCU programming directly over USB — eliminating the USB-to-Serial converter entirely.

  • Implemented bit-banging Virtual USB (V-USB) stack to handle USB protocol in firmware without dedicated USB hardware
  • Reset Interrupt Vector Table manipulation to enable clean switching between boot-mode and application mode
  • Reduced hardware BOM and simplified production programming workflow
V-USB Bit-Banging Bootloader Interrupt Vectors ATmega328P
P.03
Sinusoidal PWM Inverter Control
IISc · Real-Time Power Electronics

Algorithm development and implementation of Sinusoidal PWM switching for high-power inverters controlling AC motors, with deterministic real-time execution guarantees.

  • Direct register manipulation of 16-bit timers for precise PWM generation and dead-time insertion between complementary switch pairs
  • Implemented thermal constraints at the hardware level for protection under fault conditions
  • Achieved deterministic, hardware-level performance suitable for closed-loop motor control applications
SPWM Timer Registers Dead-Time Insertion Motor Control Real-Time
P.04
Real-Time Scheduling (FreeRTOS)
Virginia Tech · RTOS Systems

Implementation and benchmarking of scheduling algorithms for deterministic multi-task execution on embedded targets using FreeRTOS.

  • Implemented Rate Monotonic (RM), Deadline Monotonic (DM), and Earliest Deadline First (EDF) schedulers
  • Replaced priority inheritance with Priority Inheritance Protocol (PIP) and Priority Ceiling Protocol to minimize worst-case blocking times
  • Built scheduler state machine with READY, RUNNING, BLOCKED, and IDLE states using semaphores and mutexes
  • Benchmarked task execution latency, blocking times, and CPU utilization across scheduling policies
FreeRTOS RM / DM / EDF PIP / PCP Semaphores Mutexes
